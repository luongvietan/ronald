Project Goal:
Build a simple, clean, and fast web platform to connect users with service providers for events in Mauritius.

The platform should be inspired by http://mariage.net in terms of structure and user experience, but with a broader positioning:
Not only weddings, but all types of events (weddings, birthdays, corporate events, private parties, etc.)

Phase 1 Scope (MVP)

1. Homepage

* Clean, modern UI (mobile-first)
* Clear value proposition
* Sections highlighting main categories
* Simple navigation to listings

2. Categories (for MVP)

Please structure the platform with these categories:

- Photography
- Catering
- Flowers / Decoration
- Music / DJ
- Venues
- Event Planner
- Decoration

Each category should have its own page.

3. Provider Listing Page

* Grid or card layout
* Each provider should display:

* Image
* Name
* Short description
* Location (if available)
* Clean and modern design
* Prepared structure for future filters

4. Provider Profile Page

* Image gallery
* Description
* Key information (location, services)
* Contact section
* WhatsApp button (very important)
* Contact form

5. Basic Search & Filters (simple for now)

* Filter by category
* Filter by location (basic)

6. Admin Panel (very important) - Có nên dùng Sanity?

* Ability to:

* Add/edit/delete providers
* Manage categories
* Simple and easy to use (no coding needed)

7. Technical Requirements

* Built with Next.js (as discussed)
* Mobile-first design
* Fast loading performance
* Clean and scalable structure (important for Phase 2)

8. Scalability (IMPORTANT)
The structure must allow easy addition later of:

* User accounts
* Provider dashboards
* Favorites
* Paid features
